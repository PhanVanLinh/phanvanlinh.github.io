Trong _layout

cái default.html là dùng cho trang home
cái category.html là dùng cho mấy trang category
cái page.html là dùng cho mấy trang cơ bản (như trang about)
cái post.html là dùng cho trang post




Cái _site là cái mà project tự gen ra, có thể xóa được, y như cái build trong android


KHi mới install
Khi mới tải project về phải cài gem đồ vô

Xong chạy lệnh dưới để setup
bundle install

Chạy lệnh ni để build
jekyll s --watch

Build xong thì nó ra 1 cái local (ví dụ 127.0.0.1:40), chạy lên nữa là xong
